HTML Preview – 一個適用於 GitHub & BitBucket 的 .HTML 預覽工具

GitHub & BitBucket HTML Preview 是一個簡單的服務，它可以預覽任何托管在 GitHub & BitBucket 平台的 HTML 文件，並且能夠加載 CSS 樣式。從而避免直接打開鏈接看到源代碼。

這是一款在測試 Awfice – 每款不到 1KB，世界上最小辦公套件：文本編輯、表格、繪圖、幻燈片、代碼編輯器、計算器時發現的小項目，Awfice 就使用了這樣的預覽方式提供試一試服務 ߘ

GitHub & BitBucket HTML Preview

以 Awfice 為例，它直接將那個簡單的 1KB 代碼放在 GitHub 倉庫裡，代碼是這樣的：

無論是直接訪問 GitHub 的代碼頁面鏈接，還是 Raw 鏈接（Raw 按鈕是 GitHub 提供的一個功能，可以在點擊後直接顯示該文件內容的文本格式，而不是顯示整個文件的頁面），都沒辦法以 HTML 的方式顯示出來，但…這個 HTML Preview 就行。

只需要在對應的 .html 鏈接前面加上 https://htmlpreview.github.io/? 就行。

還是以 Awfice 的繪圖為例，原始鏈接：

  • 頁面：https://github.com/zserge/awfice/blob/main/edit.html

  • Raw：https://raw.githubusercontent.com/zserge/awfice/main/edit.html

只需要在這兩個鏈接前面加上 htmlpreview 即可：

  • https://htmlpreview.github.io/?https://github.com/zserge/awfice/blob/main/draw.html

  • https://htmlpreview.github.io/?https://raw.githubusercontent.com/zserge/awfice/main/draw.html
